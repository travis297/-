# 探索性数据分析 (EDA) 报告

## 1. 数据概览

### 1.1 数据集形状
- **训练集**: 150,000 条样本，31 个特征
- **测试集**: 50,000 条样本，30 个特征（不含 price）

### 1.2 缺失值分析
部分特征存在缺失值，主要集中在 `bodyType`, `fuelType`, `gearbox`。
- **训练集缺失情况**:
  - `fuelType`: 8680 (约 5.8%)
  - `gearbox`: 5981 (约 4.0%)
  - `bodyType`: 4506 (约 3.0%)
  - `model`: 1
- **测试集缺失情况**:
  - `fuelType`: 2924
  - `gearbox`: 1968
  - `bodyType`: 1504

建议后续处理：对于类别特征，可以考虑填充众数或将其作为独立类别（如“未知”）。

## 2. 目标变量分析 (Price)

### 2.1 价格分布
- **统计量**:
  - 均值: 5923.33
  - 中位数: 3250.00
  - 最小值: 11.00
  - 最大值: 99999.00
  - 标准差: 7502.00

- **分布特征**:
  从下图中可以看出，原始价格数据呈现严重的长尾分布（右偏）。这意味着大部分车辆价格较低，但有少量高价车拉高了均值。
  
  **Log 变换效果**: 对价格进行 Log 变换 (`log1p`) 后，数据分布更接近正态分布。这对线性回归模型非常重要，建议在建模时使用 Log 变换后的价格作为目标变量。

![价格分布分析](EDA_Figures/价格分布分析.png)

## 3. 特征分析

### 3.1 类别特征与价格的关系
- **bodyType (车身类型)**: 不同车身类型的价格中位数有显著差异，说明车型是定价的关键因素。
- **fuelType (燃油类型)**: 燃油类型对价格也有明显影响。
- **gearbox (变速箱)**: 自动挡和手动挡的价格分布有所不同。

![类别特征与价格关系](EDA_Figures/类别特征与价格关系.png)

### 3.2 数值特征与价格的关系
- **power (马力)**: 总体呈正相关趋势，马力越大价格越高。但在低价区也有部分高马力样本，可能是数据异常或老旧高排量车。
- **kilometer (公里数)**: 呈现明显的负相关，公里数越长价格越低。这是二手车定价中最直观的规律。

![数值特征与价格散点图](EDA_Figures/数值特征与价格散点图.png)

## 4. 训练集与测试集一致性检查

为了保证模型在测试集上的泛化能力，我们对比了训练集和测试集关键特征的分布（Kernel Density Estimation）。
- **kilometer, power, model, brand**: 从对比图中可以看到，训练集（蓝色）和测试集（橙色）的分布曲线几乎完全重合。
- **结论**: 训练集和测试集数据分布一致（独立同分布），未发现明显的 Covariate Shift，这为建立稳定的预测模型提供了良好的基础。

![训练集与测试集分布对比](EDA_Figures/训练集与测试集分布对比.png)

## 5. 总结与建议
1.  **数据预处理**: 需处理 `bodyType`, `fuelType`, `gearbox` 的缺失值。
2.  **目标变量**: 强烈建议对 `price` 进行 Log 变换以改善模型拟合。
3.  **特征工程**: `kilometer` 和 `power` 是强特征，可以考虑构造更多交互特征。
4.  **模型选择**: 鉴于非线性关系（如公里数衰减），树模型（XGBoost/LightGBM）可能比简单的线性模型表现更好，或者线性模型需要配合特征分箱/多项式特征。
