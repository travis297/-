# 二手车价格预测项目总结报告

## 1. 项目简介

本项目旨在通过机器学习算法深入挖掘二手车交易数据中的非线性规律，构建高精度的二手车交易价格预测模型。项目涵盖了从探索性数据分析（EDA）、特征工程、基线模型构建到高阶模型融合的全流程数据科学探索。研究的重点聚焦于数据预处理策略、业务逻辑特征的构造以及多模型Stacking融合架构的设计，最终成功实现了低于预设误差目标的精准定价预测。

## 2. 核心成果

* **突破精度目标**：通过实施高阶特征工程与模型融合策略，最终构建的Stacking模型在测试集上的平均绝对误差（MAE）大幅降至 **488.85**，圆满达成了误差低于500的初始项目目标。
* **全流程代码跑通**：成功建立并跑通了包含EDA分析、特征处理、8种不同基模型训练、模型交叉对比及最终高级堆叠管道（Pipeline）的完整代码工程库。

## 3. 数据说明

* **数据规模**：数据集由150,000条训练集样本（含31个特征）和50,000条测试集样本（含30个特征，无目标变量 `price`）组成。
* **缺失情况**：核心缺失值集中在类别特征，如 `fuelType`（约5.8%）、`gearbox`（约4.0%）和 `bodyType`（约3.0%）。
* **目标变量**：原始二手车价格（`price`）呈严重的长尾右偏分布，项目中采用 `log1p` 变换将其转化为近似正态分布，以保障回归模型的稳定训练。

## 4. 模型列表

项目中构建并评估了跨越三大类的多种机器学习算法：

* **线性模型**：Ridge（岭回归）、Lasso（套索回归）。
* **神经网络**：MLP（多层感知机）。
* **树形单模型与集成模型**：Decision Tree（决策树）、Random Forest（随机森林）、Extra Trees（极限随机树）、LightGBM、XGBoost。
* **融合架构**：两层 Stacking 模型。第一层（L1）集成 Extra Trees、LightGBM、XGBoost 和 Random Forest，第二层（L2）以 Ridge 回归作为元模型进行加权输出。

## 5. 模型性能对比

以平均绝对误差（MAE）为核心评价指标，各模型的验证表现如下（MAE越低越好）：

* **线性回归**：Ridge (2599.08), Lasso (2659.78) —— 无法捕捉非线性关系，表现垫底。
* **神经网络**：MLP (738.51) —— 表现尚可，但在结构化表格数据上不及顶尖树模型。
* **顶尖树模型**：XGBoost (718.74), Random Forest (660.52), LightGBM (645.91)。
* **最佳单模型**：**Extra Trees (627.88)** —— 凭借极端随机化分裂策略有效降低方差，拔得单模头筹。
* **最终融合模型**：**Stacking (488.85)** —— 综合树模型优势，相比最优单模进一步降低了约140点误差。

## 6. 特征工程影响

特征工程是本项目精度实现跨越的关键：

* **业务逻辑特征**：引入精确车龄（`car_age_years`）和折旧率（`miles_per_year`）后，XGBoost单模型的MAE从530降至492，相对提升7.1%。
* **高基数目标编码**：采用防穿越机制的5折目标均值编码（Target Encoding）处理 `model` 等特征，贡献了最大的精度红利，使整体MAE下降超30点。
* **高阶交互特征**：构建匿名特征的乘积项（如 `v0_v12_prod`），单一特征的加入便使MAE下降12点。

## 7. 项目工程结构与目录

### 7.1 核心逻辑模块概览

* **EDA与清洗模块**：包含特征分布探索、同分布检验及异常值（如99%分位数价格截断、马力封顶）处理。
* **特征工程模块**：执行多层级分组填充缺失值、业务特征提取与多维度编码。
* **单模型评估模块**：执行8种算法的本地5折交叉验证，输出特征重要性及残差分析报告。
* **Stacking 高级管道**：`advanced_model_pipeline.py`，串联所有预处理步骤并执行L1树模型与L2线性模型的端到端训练与推断。

## 8. 技术亮点

* **严谨的防数据泄露机制**：在实施目标编码时，严格采用K-Fold策略，确保当前折特征值仅由其余折的价格统计量生成，有效杜绝了高基数类别编码常引发的数据穿越与过拟合灾难。
* **精细化层级插补算法**：摒弃粗暴的全局均值/中位数填充，设计了依据物理关联（如根据同“品牌+车型”的分组众数）层级递进补全缺失值的算法，最大程度维系了客观分布特性。
* **稳健的模型架构选型**：在Stacking设计中，第一层巧妙结合了具有极端随机抗过拟合特性的 Extra Trees、高效捕捉非线性的 LightGBM/XGBoost 以及降低整体方差的 Random Forest，利用模型间差异性实现优势互补。
